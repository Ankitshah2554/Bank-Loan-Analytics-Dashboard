# Bank Loan Analytics Dashboard
## Project Overview
This Power BI project provides insights into bank loan performance and customer lending behavior.

The dashboard helps analyze:
- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average DTI Ratio
- Good Loan vs Bad Loan Analysis
- Monthly Loan Trends
- State-wise Loan Applications
- Loan Purpose Analysis
- Home Ownership Analysis
- Loan Term Analysis

## Business Insights
- Good loans contribute the majority of funded amount.
- Debt Consolidation is the most common loan purpose.
- Rent and Mortgage customers represent the largest borrower segments.
- Loan applications show an increasing trend throughout the year.
